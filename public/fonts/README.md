# Hubot Sans Font

To use the Hubot Sans font in this project:

1. Download the Hubot Sans variable font file (Hubot-Sans.woff2) from the official GitHub repository:
   https://github.com/github/hubot-sans/raw/main/fonts/variable/Hubot-Sans.woff2

2. Place the downloaded 'Hubot-Sans.woff2' file in this directory (public/fonts/).

3. Restart your development server to see the changes take effect.

Note: The font file is not included in this project by default due to licensing considerations. Please ensure you comply with the font's license when using it in your project.
